# Ingenieria de Datos | Portafolio

Bienvenido a tu espacio para registrar los proyectos, experimentos y aprendizajes de la materia de Ingenieria de Datos. Usa este sitio como bitacora viva: cada entrada debe contar que problema resolviste, como lo abordaste y que decisiones tomaste.

## Que encontraras
- Panorama del curso: objetivos, entregables y criterios de evaluacion.
- Casos practicos: notebooks, pipelines y tableros que hayas construido.
- Bitacora de aprendizaje: notas breves con hallazgos, problemas y soluciones.
- Recursos clave: guias, referencias y enlaces utiles.

## Como navegar
- Visita `Acerca de mi` para presentar tu perfil tecnico y metas del semestre.
- Crea entradas en `Portfolio` usando la plantilla para mantener formato consistente.
- Anade evidencias: capturas, enlaces a repos, metricas de calidad de datos o performance.
- Cierra cada entrada con reflexiones y proximos pasos.

## Checklist rapido antes de publicar
- Explicas el contexto y el objetivo del ejercicio?
- Incluyes decisiones de diseno (modelado, limpieza, ingesta, orquestacion)?
- Adjuntas resultados reproducibles (codigo, dataset de ejemplo, metricas)?
- Notas aprendizajes y riesgos abiertos para la siguiente iteracion?
