# UNIDAD TEMATICA 2: Calidad & Etica

Al completar UT2, el estudiante sera capaz de:

- Distinguir entre MCAR, MAR y MNAR en datasets reales
- Detectar patrones de missing data y outliers
- Aplicar estrategias de imputacion segun el contexto
- Implementar pipelines de limpieza reproducibles
- Prevenir data leakage usando validacion cruzada apropiada
- Identificar y mitigar sesgo en datasets historicos
- Evaluar fairness usando metricas estandar
- Documentar decisiones eticas en el tratamiento de datos
